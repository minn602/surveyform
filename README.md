## Survey Form

![서베이폼](surveyForm.gif)

### 소개

React.js로 만들어진 서베이 폼 입니다

### 기술스택

React.js, Redux, Styled-Components

### 시작하기

1. yarn install
2. yarn start
3. http://localhost:3000/ 에서 확인 가능합니다.

### 구현한 기능

- 폼, 옵션 추가 및 삭제 기능
- 각 인풋 상태값 저장
- 필터링에 따른 인풋 선택 기능
- submit 버튼 클릭 시 form validation 추가
